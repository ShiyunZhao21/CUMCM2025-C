##   《基于优化模型和代价敏感学习模型的NIPT检测策略与胎儿异常判定方法》


###   摘要


无创产前检查是一种通过分析孕妇外周血中胎儿之 DNA 来筛查胎儿是否存在染色体异常情况。


本文基于附件中的孕妇检测数据，使用回归方法，建立优化模型，分析了最优男胎孕妇的 BMI 分组方式和最优 NIPT 检测时间节点问题。同时我们基于机器学习方法，给出了女胎染色体异常的判断方法。


针对问题一，首先进行数据预处理，基于 GC 含量，Y 染色体浓度等数据剔除了异常检测数据，并对同一孕妇同一时间的检测数据取平均来降低误差。在此之后，基于处理后的数据对孕妇群体的
Y 染色体浓度，BMI，年龄等指标进行了描述性统计与正态性检验。随后，计算得到了孕妇各个指标的 Spearman 相关性系数以评价各变量之间相关性，并利用线性回归方程得到了这些指标和 Y
染色体浓度的定量关系。


针对问题二，首先利用函数来量化一个孕妇在不同时间进行检测的风险，以此确立了优化模型的目标函数，即使所有孕妇的风险之和最小化。随后，我们确定了优化模型的决策变量，即 BMI 分组的各组端点及每组 BMI 孕妇的检测时间，然后，基于问题一得到的 Y 染色体浓度与 BMI 和孕周数之间线性关系，确定了每组检测效果的达标条件，并添加了一系列约束条件使模型合理化。最后，我们利遗传算法对模型进行了求解，并通过添加扰动项的方法模拟检测误差，分析其对优化结果及孕妇风险的影响。


针对问题三，首先利用函数来量化孕妇群体检测的风险大小和准确率，以此确立优化模型的目标函数，也就是实现孕妇群体风险最小，达标率最高。随后，我们确定了优化模型的决策变量，即BMI 分组的各组端点及每组 BMI 孕妇的检测时间。然后，考虑到 Y 染色体浓度受多因素影响，基于问题一得到的 Y 染色体浓度与 BMI、年龄、孕周数之间线性关系，在线性关系中加入误差项，确定了每组检测效果的达标条件，并添加了一系列约束条件使模型合理化。最后，我们利用遗传算法对模型进行了求解，并通过改变误差项大小评估误差优化结果的影响。


针对问题四，我们同样先进行了数据预处理，之后为了得到综合众多因素给出女胎是否异常的判定方法，我们建立了机器学习中的逻辑回归模型。针对疾病检测问题中漏诊代价巨大的特点，我们引用了代价敏感学习对模型进行优化，同时调整参数使得模型可以有效地筛选出了染色体异常胎儿，提高了召回率，同时得到了各个影响因素对胎儿染色体是否异常的特征系数。最后，我们根据以上结果得出了判断是否异常的具体方案。


关键词：多元线性回归模型；优化模型；遗传算法；机器学习（逻辑回归）；代价敏感学习
